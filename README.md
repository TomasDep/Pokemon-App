First project with next.js with use PokeminAPI

## Technologies

-   Next.js v13
-   NextUI v2.1.7
-   Tailwind v3
-   TypeScript
-   React v18

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
